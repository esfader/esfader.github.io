Encriptador de texto

Este código es una aplicación de encriptación y desencriptación de texto utilizando JavaScript.

La aplicación tiene dos funciones principales: encrypt y decrypt. La función encrypt toma un texto como entrada y lo encripta utilizando un diccionario que reemplaza cada una de las vocales por una secuencia de caracteres específica. La función decrypt hace lo contrario, utilizando el mismo diccionario para desencriptar el texto.

El código también incluye una expresión regular que se utiliza para validar si el texto contiene caracteres no permitidos (mayúsculas y caracteres con tilde). Si el texto cumple con la expresión regular, se muestra una alerta y no se realiza la encriptación o desencriptación.

Además, el código incluye dos funciones que manejan la lógica de la aplicación: encriptar y desencriptar. Estas funciones toman como entrada un evento y utilizan los métodos encrypt y decrypt para encriptar o desencriptar el texto, respectivamente. También validan si hay texto para encriptar o desencriptar y muestran un mensaje de resultado en pantalla.

El código también incluye una función copiarPortapapeles que utiliza el API de portapapeles de JavaScript para copiar el texto encriptado o desencriptado al portapapeles del usuario.

Calculadora

Este código define una calculadora que permite realizar operaciones matemáticas básicas, como sumar, restar, multiplicar y dividir. También permite calcular la raíz cuadrada de un número y calcular una potencia.

La función addNumber se encarga de añadir un número a la pantalla de la calculadora. Si la pantalla muestra el valor "0", reemplaza ese valor por el número que se está añadiendo. Si la pantalla ya muestra un número distinto de "0", simplemente concatena el nuevo número al final.

La función calculateResult se encarga de calcular el resultado de la operación matemática que se ha estado construyendo. Primero, almacena el valor actual de la pantalla en una variable llamada calculation, que luego se usará para evaluar la expresión matemática. Luego, asigna al valor de la pantalla el resultado de evaluar calculation y muestra el resultado en un elemento de la página web con la clase "result". Si el resultado es "NaN" (not a number) o "Infinity", significa que la operación es inválida y se reinicia la calculadora. Finalmente, se ejecuta la función limpiarDisplay después de 3 segundos para borrar el resultado de la pantalla.

La función handleOperator se encarga de añadir el operador seleccionado a la operación matemática en construcción. Dependiendo del operador seleccionado, se añade a calculation el valor actual de la pantalla seguido del operador correspondiente. Luego, se reinicia la pantalla a "0".

Las funciones calculateSquareRoot y calculatePower se encargan de calcular, respectivamente, la raíz cuadrada de un número y una potencia. Ambas funciones piden al usuario que ingrese el valor del exponente en el caso de calculatePower. Luego, asignan el resultado de la operación a la pantalla.

Por último, la función limpiarDisplay se encarga de reiniciar la pantalla de la calculadora a "0".
